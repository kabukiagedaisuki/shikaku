| 内容 | 語呂 |
|:----|:-----|
| 予定死亡率 | 率が低いと（死亡する人が少ないので）保険料は下がる |
| 予定利率<br> * 保険会社があらかじめ見込んでいる運用利回り | 率が高いと（収益が多いので）保険料は下がる |
| 予定事業費率<br> * コスト | 率が低ければ（コストが低いので）保険料は下がる |

| 養老保険<br> * 死んだら死亡保険金、満期に生きてたら満期保険金がもらえる |  |
| 定期保険特約付終身保険 | 定期保険＋終身保険により死亡保険金を厚くする |
| 変額保険<br> * 株や債券の運用益を保険金などに反映させる<br> * 特別勘定の口座で運用される<br> * 最低保証（基本保険金）がある<br> * 解約返戻金や満期保険金に最低保証はない |  || 特約 | 180日や6ヶ月という事例が多い |
